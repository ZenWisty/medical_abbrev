# medical_abbrev

---

### 项目意义

让像我一样的半外行理解从身边的内容中构建术语集合，理解一些医疗术语以及他们使用的场景。主动从真实来源（视频、文档、网络）中提取医学术语，建立自己的术语库，并通过注释理解这些术语出现的实际场景。

最终产物是医疗术语语料库 + 场景化注释。
本项目中的内容仅供学习用途

---

### 使用的技能（Skills）

项目中使用的简单技能：
#### 1. `.claude/skills/video_speechtxt_translate`
将视频的语音字幕文件（格式为 `00:start|end|text`）从英文翻译为中文。保留时间戳和格式，适用于医学讲座、教程视频等含有大量专业术语的内容。
#### 2. `.claude/skills/filter-out-medterm`
医学术语过滤技能。输入双语字幕文件（`00:start|end|text` 格式），输出只保留含有临床医学术语的行（疾病名称、药物名称、医疗操作、临床指标等）。自动排除社工对话、心理支持、音乐歌词、寒暄闲聊等非临床内容。
#### 3. `.claude/skills/video_speechtxt_shrink`
将翻译后的字幕文件压缩为话题聚焦的摘要格式。按主题分组，用简洁的摘要行替代原始时间线，适合快速浏览大量内容或提取关键知识点。

### 案例介绍

#### examples/医疗短视频提取

**来源**：医疗短剧 字幕中的场景内医疗术语提取。

**处理流程**：
1. `video_speechtxt_translate` 将英文字幕翻译为中文
2. `filter-out-medterm` 过滤出含医疗术语的行
3. 搜索补充术语的详细解释

**输出文件**：
- `ThePIT_E01_translate_medfilt.md` — 原始双语字幕过滤版
- `ThePIT_E01_translate_medfilt_comments.md` — 带详细注释版

**内容示例**（含注释的医疗术语）：
```
00:04:51,460|00:04:54,860|做美普罗斯托米，还有一个小肠梗阻
#### 小肠梗阻 (SBO)：常见外科急症，因肠内容物无法通过小肠所致。病因包括术后粘连（最常见）、疝气、肿瘤、克罗恩病等。CT是诊断金标准。
```

---

#### examples/有源器械注册

**来源**：医疗器械设计开发及注册方法--mindray。

**处理流程**：
视频原文 + 关键段落截图 + 知识点提炼。侧重医疗器械法规、注册流程、技术文档等与从业者直接相关的内容。

**输出文件**：
- `BV1GD_笔记.md` — 含时间戳、原文引用、重点提炼、截图
- `screenshots/` — 关键帧截图

**内容示例**：
```
00:01:37,800|00:01:38,820|有源医疗器械
#### 定义：相对于无源器械，需要使用电能等驱动的医疗器械
#### 举例：灰阶、马力阶级、病人监护设备等
```
